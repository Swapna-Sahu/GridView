# Test Assignment

Build a grid view using all the JSON data provided

## Features

1. Pagination
2. A search function
3. A ‘no results found’ state on search
4. Make it accessible and keyboard friendly
5. Add new tile

## Technology used 
ReactJs , React Hook, Javascript, CSS

## Project setup
1. Clone the repo
2. run npm install
3. run npm start
